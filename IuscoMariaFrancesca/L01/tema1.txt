Folosesc Microsoft 365 atat la locul de munca, cat si pentru uz personal.
Microsoft 365 este o aplicatie cloud de tipul software-as-a-service si este conceputa pentru a oferi ajutor utilizatorilor, 
astfel incat acestia sa poata realiza mai multe cu aplicatiile Office inovatoare, cu servicii inteligente de cloud si cu securitate de nivel superior.
Pe langa aplicatiile precum Word, Excel, PowerPoint, prin Microsoft 365 se poate stoca, accesa si partaja pana la 1TB de spatiu de stocare online pentru fiecare utilizator, 
se poate pastra echipa unita prin chat-ul de grup, intalniri online si apelare prin Microsoft Teams, se pot utiliza emailul si calendarul pentru a contacta clientii si 
colegii de serviciu oriunde ar fi acestia.
